# Лабораторная работа №3: Условные и циклические алгоритмы на C++

**Цель работы**: Изучить разработку и реализацию программ с использованием условных и циклических алгоритмов.

## Задание
1. Определить, является ли введённое число положительным, отрицательным или нулём (с помощью `if/else`).
2. Подсчитать сумму элементов массива от 1 до 10 (с помощью цикла `for`).
3. Распределить человека в команду по номеру класса (1 – зелёные, 2 – красные, 3 – синие, 4 – жёлтые) с помощью `switch`.
4. Выводить частичные суммы элементов массива от 1 до 10 до тех пор, пока общая сумма не превысит 32 (с помощью цикла `do while`).

Все программы написаны на C++ и включают базовую обработку некорректного ввода.

## Требования
- Компилятор с поддержкой C++17 (например, `g++` или `clang++`).
- Утилита `make`.
- (Опционально) Для сборки отчёта — `xelatex` (входит в дистрибутивы TeX Live, MikTeX).

## Сборка и запуск
Все команды выполняются из корневой папки проекта.

### Сборка всех программ
```bash
make all
```
Исполняемые файлы `point1`, `point2`, `point3`, `point4` появятся в каталоге `bin/`.

### Запуск конкретной программы
```bash
make run_point1   # задание 1 (знак числа)
make run_point2   # задание 2 (сумма массива)
make run_point3   # задание 3 (команды)
make run_point4   # задание 4 (частичные суммы)
```

### Запуск всех программ последовательно
```bash
make run_all
```

### Очистка сгенерированных файлов
```bash
make clean
```

### Полная пересборка
```bash
make rebuild
```

### (Опционально) Сборка отчёта в PDF
```bash
make report
```
Для этого требуется установленный XeLaTeX. PDF будет создан в папке `report/` как `lab3.pdf`.
